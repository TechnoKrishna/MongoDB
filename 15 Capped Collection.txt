# CAPPED COLLECTION

* A CAPPED COLLECTION IN MONGODB IS A FIXED-SIZE COLLECTION THAT AUTOMATICALLY OVERWRITES ITS OLDEST DATA WHEN THE SPECIFIED SIZE LIMIT IS REACHED. 
* IT MAINTAINS INSERTION ORDER AND OFFERS HIGH-PERFORMANCE WRITES. 
* MAKING IT IDEAL FOR USE CASES LIKE LOGGING, CACHING, AND REAL-TIME DATA STREAMS.

-> db.createCollection("order_logs",{capped:true, max:4, size:100000})
